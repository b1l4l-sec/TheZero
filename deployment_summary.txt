TROUBLESHOOTING GUIDE
───────────────────────────────────────────────────────────

Issue: Permission denied
Fix: chmod +x Setup.sh StartUp.sh theZero.py

Issue: Python not found
Fix: sudo apt-get install python3 python3-pip python3-venv

Issue: Setup fails
Fix: rm -rf venv && ./Setup.sh

Issue: Port 5000 in use
Fix: sudo lsof -ti:5000 | xargs kill -9

Issue: Import errors
Fix: source venv/bin/activate && pip install -r SpecterVision/requirements.txt

Issue: Camera not working
Fix: Grant browser permissions, try different browser
