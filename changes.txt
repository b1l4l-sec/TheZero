═══════════════════════════════════════════════════════════
  HOW TO USE
═══════════════════════════════════════════════════════════

INSTALLATION (First Time)
───────────────────────────────────────────────────────────
cd TheZero
chmod +x setup.sh thezero.py
./setup.sh

LAUNCHING THEZERO
───────────────────────────────────────────────────────────
./thezero.py

Or:
python3 thezero.py

Note: Virtual environment activates automatically

CTRL+C BEHAVIOR
───────────────────────────────────────────────────────────
✓ In Main Menu: Exits TheZero completely
✓ In Sub-Tool: Returns to main menu
✓ In Documentation: Returns to main menu
✓ No more complete shutdowns from sub-tools

WORKFLOW EXAMPLE
───────────────────────────────────────────────────────────
1. ./thezero.py
2. Select [1] - Crypto Suite
3. Port scanner starts
4. Press Ctrl+C → Returns to main menu
5. Select [2] - SpecterVision
6. Press Ctrl+C → Returns to main menu
7. Select [0] → Exits completely

Virtual Environment Activation:
- check_and_activate_venv() function added
- Detects if running in venv (sys.prefix vs sys.base_prefix)
- If not in venv, uses os.execv() to restart with venv Python
- All subprocess calls now use venv Python interpreter

Keyboard Interrupt Handling:
- launch_crypto_suite() - try/except KeyboardInterrupt
- launch_spectervision() - try/except KeyboardInterrupt
- Documentation menu - nested try/except
- Main menu - only place where Ctrl+C exits
