═══════════════════════════════════════════════════════════
  WORKFLOW
═══════════════════════════════════════════════════════════

1. INSTALLATION
   └─> ./Setup.sh
       ├─> Install system packages
       ├─> Create virtual environment
       ├─> Install Python dependencies
       └─> Set permissions

2. LAUNCHING
   └─> ./StartUp.sh
       ├─> Activate venv
       └─> Run theZero.py

3. MAIN MENU
   └─> theZero.py
       ├─> [1] Launch Crypto&portScan/thezero.py
       ├─> [2] Launch SpecterVision/spectervision.py
       ├─> [3] Show Documentation
       └─> [0] Exit

4. CRYPTO SUITE (Option 1)
   └─> Crypto&portScan/thezero.py
       ├─> [1] Port Scanner
       ├─> [2] Cryptography x64
       ├─> [3] Caesar Cipher
       ├─> [4] Password Generator
       └─> [5] Back

5. SPECTERVISION (Option 2)
   └─> SpecterVision/spectervision.py
       ├─> [1] Launch Server → Browser → Camera
       ├─> [2] ngrok Tunnel → Remote Access
       ├─> [3] Open Captures
       ├─> [4] Check Dependencies
       └─> [0] Back

═══════════════════════════════════════════════════════════
  TECHNICAL DETAILS
═══════════════════════════════════════════════════════════

Port Scanner:
- Protocol: TCP
- Method: Multi-threaded socket connections
- Timeout: 1 second per port
- Range: 1-65535

Camera Capture:
- Resolution: 640x480 (default)
- Format: JPEG
- Quality: 85%
- Interval: 3 seconds
- Storage: Base64 → Binary

Emotion Detection:
- Model: MediaPipe FaceMesh
- Runtime: TensorFlow.js (WASM)
- Max Faces: 5
- Emotions: 7 (Happy, Sad, Angry, Surprised, Fearful, Disgusted, Neutral)

Motion Detection:
- Method: Frame differencing
- Threshold: 5% motion for alert
- Pixel Threshold: 30 RGB difference
- Overlay: Real-time red highlighting
